#  PAC-MAN EN CONSOLA
## Descripción

Este proyecto es una implementación del clásico juego **PAC-MAN** para consola, desarrollado en Java como parte de la Práctica 1 del curso de Introducción a la Programación y Computación 1. El juego permite configurar un tablero personalizable, colocar elementos aleatorios y controlar a PAC-MAN para recolectar premios mientras se evitan fantasmas y paredes.

###  Características Principales
Tablero configurable, elementos aleatorios, Sistema de puntajePremio simple, Vidas, Bordes infinitos, Sistema de pausa, Historial


## Objetivo del Juego

Controlar a **PAC-MAN** (`<`) para recolectar **TODOS los premios** del tablero mientras se evitan:

- **Paredes (`X`)**,  No se pueden atravesar
- **Fantasmas (`@`)**, Quitan 1 vida al contacto

**Ganas** si recolectas todos los premios. **Pierdes** si te quedas sin vidas.
